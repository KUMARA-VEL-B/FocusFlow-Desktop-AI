# ⚡ FocusFlow Desktop AI

FocusFlow is a clean, hyper-responsive, and modern Daily Task Manager built as a **Progressive Web App (PWA)**. Designed for simplicity and flow state, it helps you manage your day securely and effectively without unnecessary database fluff.

## ✨ Features

- **✅ Core Task Management**: Add, effortlessly edit, completely complete, or gracefully delete daily tasks.
- **🔄 Day Rollover & History**: Unfinished tasks seamlessly transition to "Pending" rules the next day, and finished tasks are securely archived to your localized "History". 
- **⚖️ Native Drag & Drop**: Physically prioritize and reorder your tasks instantly on your daily list.
- **🏷️ Priority Flags**: Mark your tasks creatively with explicit priority tags (High 🔴, Medium 🟡, Low 🟢).
- **🛡️ Accidental Deletion Safeties**: Deleted a task by mistake? Use the 5-second **Undo Toast** mechanism to restore it immediately safely.
- **📥 PWA Installation**: FocusFlow dynamically tracks your installation capabilities! With a dedicated "📲 Install App" indicator cleanly placed in the Navbar, instantly install this app natively via supported browsers completely seamlessly.  
- **📊 Real-time Progress Tracking**: Watch dynamic gradient bars visually showcase your productivity and completion percentage as things get done.
- **🌤️ Smart Weather Integration**: Integrated via `wttr.in`, automatically locating coordinates to give you accurate environment stats alongside subtle, themed dynamic background weather updates. 
- **🌗 Light/Dark Optimization**: Fully robust native theme toggling capability integrated with CSS variable architectures.

## 🛠️ Built With

- **HTML5** & **CSS3** (Fluid variables, Glassmorphism elements, CSS-only micro-animations)
- **Vanilla JavaScript** (Zero bloated frameworks, pristine modular functional programming!)
- **Service Workers** (Aggressive asset caching via `sw.js` for offline PWA reliability)
- **Web Storage API** (Secure & entirely offline `localStorage` manipulation logic)
- **Geolocation API** (Accurate weather mapping) 

## 🚀 Getting Started

To get a local copy running, simply follow these completely lightweight steps!

### Prerequisites

You need a simple HTTP server (or just a basic modern web browser without intense strict CORS policy interference)!
We highly recommend using `npx serve` or standard `Live Server`.

### Installation 

1. **Clone the repo**
   ```bash
   git clone https://github.com/your-username/focusflow-pwa.git
   ```
2. **Navigate to the directory**
   ```bash
   cd focusflow-pwa
   ```
3. **Launch a local server**
   ```bash
   npx serve .
   ```
4. **Open your browser** and visit `http://localhost:3000` 
5. Use the explicit `📲 Install App` button to fully transform this into a desktop application utilizing Chromium/Edge architecture globally!
